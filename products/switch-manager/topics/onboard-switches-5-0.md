# Onboard switches

Use this procedure to onboard switches into Switch Manager.

## Steps

1. Open **Inventory > Switches**.
2. Select **Add switches**.
3. Upload the device list.
4. Assign the switches to a site.
5. Review the site compliance preview before submitting.

## Verification

Confirm that the switches appear in the site inventory.
