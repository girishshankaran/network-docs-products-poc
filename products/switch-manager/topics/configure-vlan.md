# Configure VLANs

Use this procedure to configure VLANs for switch ports.

## Steps

:::version only="4.0"
1. Open **Switching > VLANs**.
2. Select **Create VLAN**.
3. Assign ports manually.
:::

:::version from="5.0"
1. Open **Switching > Network Segments**.
2. Select **Create Segment**.
3. Add VLAN details and assign ports.
:::

## Verification

Confirm that assigned ports show the expected VLAN membership.
