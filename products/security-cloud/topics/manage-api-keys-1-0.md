# Manage API keys

Use this procedure to create and rotate API keys for integrations.

## Steps

1. Open **Integrations > API keys**.
2. Select **Create key**.
3. Choose the required integration scope.
4. Store the generated key securely.

## Verification

Confirm that the integration can authenticate with the new key.
