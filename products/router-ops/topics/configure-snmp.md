# Configure SNMP monitoring

Use this procedure to configure SNMP monitoring for routers.

## Steps

1. Open **Monitoring > SNMP**.
2. Add the collector address.
3. Select the router group.
4. Save the monitoring configuration.

## Verification

Confirm that the collector receives SNMP telemetry.
